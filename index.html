How this repo works

Simple structure, separate paths

The root page acts as a directory. Each project lives in its own folder with its own `index.html`, which means every mini-app can be visited directly without needing a separate repository.

  • `/` is the landing page for the collection.
  • `/mylifeinweeks/` is the first published mini-project.
  • `/flappybird/` is the arcade game mini-project.
  • `/sudoku/` is the touch-friendly logic game mini-project.
